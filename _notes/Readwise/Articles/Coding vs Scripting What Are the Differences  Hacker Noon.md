## Highlights
- The main differences are that coding lets you create a particular program, where scripting lets you control a program that you or somebody else had created with code.
- Someone needs to think about how that app is going to work and look, that person is called a designer.
- there is a person that implements the features that the designer designed, that person is a coder.
- For example, you made a website that just displays some images and text, that website is static, there is nothing that’s changing on that website.
  On the other hand, you use some scripting knowledge and make images and text on that website move around. Now that website isn’t static anymore, it’s dynamic, there are things that are changing on that website.
